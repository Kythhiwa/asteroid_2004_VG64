# asteroid_2004_VG64

1. Преобразовать коориданты обсерваторий из сферических в декартовы (ITRS)
2. Преобразовать коориданты обсерваторий из ITRS в GCRS
3. Интерполировать коориданты центра Земли( Видимо взять с какого-то источника)
4. Прибавить коориданты обсерваторий(GCRS) к коордианатам центра земли(BCRS) получатся
    координаты обсерваторий в BCRS
5. Методом Гаусса по нескольким наблюдениям найти приблизительное начальное состояние астероида
6. Интегрирую движение астероида с моими начальными данными
7. Используя МНК полочуем модельные Ra, Dec астероида
   Используем поправку на световой промежуток перед сравнением модельных данных
